## 📌 Descripción

Este proyecto es una **simulación educativa de captura y análisis de eventos de teclado** con fines de aprendizaje en programación y manejo de eventos de entrada.

Permite entender cómo funcionan:
- Eventos del teclado
- Registro de entrada
- Procesamiento de datos en tiempo real
- Exportación de datos para análisis

---

## 🧪 Uso responsable

Este proyecto debe ejecutarse únicamente en:

- Máquinas virtuales
- Entornos de pruebas locales
- Laboratorios de aprendizaje

Usa este software con responsabilidad, ética y respeto por la privacidad.

---

## 🛠️ Tecnologías usadas

- Python
- Manejo de eventos de entrada
- Exportación de datos (Excel / CSV)

---

## 📚 Objetivo educativo

El objetivo es aprender cómo funcionan los sistemas de entrada en aplicaciones, no la recolección de datos reales de usuarios.
